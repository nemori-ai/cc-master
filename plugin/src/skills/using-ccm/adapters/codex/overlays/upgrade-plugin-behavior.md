Codex 的 plugin 升级 adapter 会从本机已安装的 cc-master 包读取 adapter 产物，刷新本地 Codex marketplace/plugin 注册，并按 `plugin/dist/codex` 中的 skills + hooks 进行更新；入口命令以 `\$cc-master-*` skill 调用为主。传 `--all-harnesses` 时先跑本机 harness inventory，只对已安装且支持 `pluginDistribution` 的 harness 执行各自 adapter；Codex adapter 不触碰 Claude Code settings / marketplace。
