# Name and Trademark Policy / 名称与标识政策

This policy covers the `cc-master` project name and the project's logos and other
brand identifiers, if any. These identifiers are used to distinguish the
official project. This document does not claim that any mark is registered.

本政策适用于 `cc-master` 项目名称，以及项目使用的 logo 和其他品牌标识（如有）。这些标识
用于区分官方项目。本文不声称任何标识已经注册为商标。

## Permitted uses / 允许的使用

You may:

- refer truthfully to the official project;
- state accurately that your work is based on, compatible with, or derived from
  cc-master;
- use the name in links, citations, reviews, tutorials, and other descriptive
  references that do not imply endorsement.

你可以：

- 如实指代官方项目；
- 准确说明你的作品基于、兼容或衍生自 cc-master；
- 在链接、引用、评测、教程和其他描述性场景中使用该名称，但不得暗示官方背书。

## Uses that require written permission / 需要书面许可的使用

Without separate written permission from the relevant rights holders, you may
not:

- name or brand a fork, derivative, product, company, or service `cc-master`, or
  use a confusingly similar name or logo;
- present any unofficial distribution, product, service, website, account, or
  organization as official, approved, sponsored, or endorsed by cc-master;
- remove or obscure attribution in order to make the project appear to be your
  own original product.

未经相关权利人另行书面许可，你不得：

- 使用 `cc-master` 或容易造成混淆的近似名称、logo，为 fork、衍生版本、产品、公司或服务
  命名或包装；
- 把任何非官方发行版、产品、服务、网站、账号或组织宣传为 cc-master 官方、批准、赞助或
  背书的对象；
- 删除或遮蔽来源信息，以便把本项目伪装成你自己的原创产品。

If you publicly distribute a modified version, use a clearly distinct name and
branding, preserve the notices required by the software license, and describe
the relationship to cc-master accurately.

如果你公开分发修改版，请使用明显不同的名称和品牌，保留软件协议要求的声明，并准确描述它
与 cc-master 的关系。
